1. 音效
	1.1. 更新公告
		B站视频发布时有81个音频文件，现已更新至276个。
		使用新版音频文件包前请先删除旧版音频文件。
	1.2. 使用方法
		您可使用reg文件导入声音方案，然后任务栏音量图标右键→声音（Win11：声音设置→更多声音设置），
		声音方案选择201或202（101是UP主自用的，包含一些其他的音效）。
		安全提示：导入前，可先用文本编辑器打开它来检查代码。
	1.3. 我懒得去一个个设定音效。
		UP主自用声音方案在视频简介→百度网盘链接→原神游戏音效选集→相关附件→配置文件→此文件夹下的reg文件。
		下载并打开之，系统会提示导入注册表，导入后打开「声音」控制面板切换声音方案即可。
	1.4. 想要中配（或英配韩配）派蒙的音量调节语音。

2. 鼠标指针
	2.1. 更新公告
		「CZJSTMC」现已更名为「STMC」。
		2023/08/06 大型更新：
			此次更新为大型更新。更新后的作品与以前视频演示中的鼠标指针差异较大。
			改进了很多，也废除了很多质量欠佳的鼠标指针，且更改了安装配置。
			若您使用过旧版本的鼠标指针，请 (参考PDF教程) 卸载后再重装。
			若想要使用一些已被废除的旧版鼠标指针，请下载旧版存档 (Archive 文件包)。
	2.2. 使用
		2.2.1. 使用方法
			请查看「STMC保姆级使用教程」，或进入「STMC」文件夹查看「INF文件使用说明」。
		2.2.2. Win11 能用吗？
			从 WinXP 到 Win11 均可使用。
		2.2.3. 右键菜单没有「安装」。
			① Win11 系统需要先点击「显示更多选项」调出经典右键菜单。
				可以通过修改注册表让 Win11 默认使用经典右键菜单，具体方法请自行搜索。
			② 请留意您是否【单击选中了】inf文件然后再点右键。
				据我观察他人使用电脑的经验，很多人忘记先左键选中文件然后再右键，导致他们相当于在空白处调出右键菜单。
				这种右键菜单是针对当前文件夹，而不是您想选中的文件，自然不会出现诸如安装inf的选项。
		2.2.4. 如何卸载？
			运行「control mouse」打开「鼠标 属性」窗口，点击「指针」选项卡，逐个删除所有「STMCS」方案。
			然后打开资源管理器，删除 C:\Windows\Cursors\STMC
	2.3. 特定鼠标指针
		2.3.1. 【已过时】想要纳西妲长按E技能采花的小手。

3. 字体替换
	3.1. 使用建议
		这款字体（汉仪文黑-85W）其实并不适合日常用作系统UI字体来显示大量小号文字，建议尝尝鲜即可，用腻了就换回去。
		推荐将其用作桌面歌词字体，效果很不错。
	3.2. MacType 的风险
		MacType 软件功能涉及 Windows 系统底层，因此具有一定的不稳定性，但风险程度相对较低，可以放心。
		不过切勿在较新版本 Windows 上使用旧版本 MacType，Win7 至 Win10 之间搭配最新版本的 MacType 基本没问题。
		万一安装使用 MacType 导致系统崩溃，可重启按 F8 进入 Windows 安全模式然后卸载 MacType。
		若 MacType 在您的电脑上效果异常，请参考下面的解决方案。
	3.3. 使用 MacType 后，浏览器里简体中文粗细不一。
		简体中文粗细不一多半是因为浏览器优先调用了某个日文字体，而日文字体缺少很多简体中文汉字，所以很多简体字会被 fallback 成另外一个字体。
		解决方案：
		① 浏览器设置→语言优先级，确保中文在日文前面；
		② 浏览器设置→字体，将西文（拉丁文）、中文、日文字体全部改为「汉仪文黑-85W」（可能显示为「SDK_JP_Web 85W」）；
		③ 浏览器设置→字体→无视网页指定的字体（强制网页使用此处设定的字体）；
		④ 打开 MacType...Genshin...ini，将第363至367行（Gothic...）开头的分号全部去掉；
		⑤ 打开 MacType...Genshin...ini，在第273行「ExcludeSub」下面添加「Chrome.exe」「Firefox.exe」或其他（默认已经添加）；
		⑥ 禁用 MacType。（捂脸）
